# Ecosystem & Tools

Beyond basic save editing, RotoDex includes advanced tools and ecosystem features for competitive players, collectors, and casual users alike.

---

## RotoDex Analyzer

Advanced legality explanation. Instead of simply displaying "Legal" or "Illegal", the Analyzer breaks down exactly *why* a Pokémon is legal or illegal.

**Displays**:
- Origin Game
- Encounter Method
- Ball Validation
- Move Validation
- Ribbon Validation
- PID Validation
- Generation Path

---

## RotoDex Vault (Future)

An optional, offline-by-default local Pokémon archive.

**Purpose**:  
A secure place to store and organize Pokémon outside of save files.

**Structure**:
```text
Vault
 ├── Boxes
 ├── Pokémon
 ├── Teams
 └── Collections
```

---

## RotoDex Events

A searchable event database synchronized from PKHeX resources.

**Includes**:
- Mystery Gifts
- Historical distributions
- Event Pokémon

---

## RotoDex Team Builder

Tools for crafting and analyzing competitive teams.

**Features**:
- Competitive teams
- Team import/export
- Team storage
- Team analysis

---

## Backup System

Every save modification should support automatic and manual backups.

**Workflow**:
1. Save Opened
2. Auto Backup Created
3. User Edits
4. Save Written

---

## AI Layer (Future)

An optional AI-driven assistant that enhances the user experience but **never** replaces core editing functionality.

**Capabilities**:
- Build Team
- Explain Legality
- Suggest Improvements
- Analyze Pokémon
