# RotoDex Applications

RotoDex is a multi-platform ecosystem. While `RotoDex.Core` provides the shared logic, the ecosystem is delivered to users through multiple distinct applications.

---

## RotoDex Desktop
**Platform**: Windows  
**Name**: RotoDex Desktop  

The primary, fully-featured save editing application designed as a modern replacement for the traditional PKHeX UI.

**Features**:
- Full save editing
- Pokémon editing
- Trainer editing
- Box editing
- Item editing
- Batch editor
- Legality analysis
- Multi-tab support
- Save comparison
- Undo history
- Backup system
- Plugin support

---

## RotoDex Mobile
**Platform**: Android  
**Name**: RotoDex Mobile  

A mobile companion focused on fast loading, touch-friendly design, and full offline operation.

**Features**:
- Open saves
- Pokémon editing
- Box management
- Legality checking
- Import/export Pokémon files

---

## RotoDex Web
**Platform**: Web Browser  
**Name**: RotoDex Web  

A browser-based suite of Pokémon tools that shares the same core logic as the desktop and mobile apps.

**Features**:
- Save inspection
- Legality reports
- Pokémon viewer
- Team builder
- Event database

---

## RotoDex Bot
**Platform**: Discord  
**Name**: RotoDex Bot  

A Discord bot that provides remote access to RotoDex systems directly from your server.

**Commands**:
- `/check` - Legality reports
- `/team` - Team analysis
- `/generate` - Pokémon generation
- `/compare` - Pokémon inspection
- `/eventdex` - Resource lookups
