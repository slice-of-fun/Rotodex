Welcome to **RotoDex**, a modern Pokémon save editing platform.

## Modules
- **RotoDex Desktop** - A full-featured WinForms application for deep save editing, legality checking, and batch processing.
- **RotoDex Web** - A Blazor WASM client for reading and sharing saves and teams on the go.
- **RotoDex Bot** - A Discord bot that allows remote save manipulation and team analysis.
- **RotoDex Mobile** - An Android client for on-the-go editing.

## Features
- **AI Integration**: Powered by Groq to give you plain-language legality explanations and competitive team building suggestions!
- **Auto-Syncing**: Always up-to-date with the upstream PKHeX core.
- **Advanced Legality Analyzer**: Don't just know if it's illegal, know *why*.
